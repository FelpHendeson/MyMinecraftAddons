# myAddons

Este repositório é um workspace para organizar múltiplos projetos de addons para Minecraft Bedrock Edition.

O objetivo é manter ideias, documentação, referências, templates, ferramentas e projetos independentes em uma estrutura clara, sem misturar arquivos de addons diferentes.

## Estrutura

- `addons/`: contém os projetos independentes de addons.
- `addons/nome_do_addon/`: pasta própria de cada addon, com documentação, packs, assets e arquivos de distribuição.
- `shared/`: espaço reservado para referências, templates e ferramentas reutilizáveis entre projetos.
- `docs/`: documentação geral do workspace, incluindo ideias, fluxo de trabalho e prompts.

Nenhum projeto futuro deve misturar arquivos diretamente na raiz do repositório. A raiz deve guardar apenas documentação geral e diretórios de organização.
