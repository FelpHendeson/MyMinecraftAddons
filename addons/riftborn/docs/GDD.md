# GDD - Game Design Document

## Visão geral

Riftborn: Crônicas do Novo Mundo é um addon de RPG para Minecraft Bedrock Edition inspirado em isekai, fantasia medieval e dark fantasy leve.

O jogador assume o papel de um Riftborn, alguém transportado por uma fenda dimensional para um mundo antigo, fragmentado e perigoso.

## Fantasia do jogador

O jogador deve sentir que chegou a um novo mundo e precisa crescer dentro dele, descobrindo relíquias, enfrentando ameaças, encontrando guildas e evoluindo sua identidade como aventureiro.

## Pilares

- Progressão gradual.
- Exploração recompensadora.
- Equipamentos com identidade.
- Lore fragmentada.
- Ameaças crescentes.
- Sistemas simples antes de sistemas complexos.

## Escopo inicial

A fase inicial deve preparar a base técnica e conteúdos simples. Bosses, grandes dungeons, skill tree, sistema completo de classes, UI customizada complexa e scripts avançados ficam fora do escopo inicial.
