# Riftborn: Crônicas do Novo Mundo

`Riftborn: Crônicas do Novo Mundo` é um projeto de addon para Minecraft Bedrock Edition com foco em RPG, isekai, fantasia medieval e dark fantasy leve.

A premissa é que o jogador é um Riftborn, alguém trazido por uma fenda dimensional para um mundo antigo, fragmentado e cheio de criaturas, relíquias, guildas e ameaças.

## Objetivo

Transformar gradualmente o Minecraft Bedrock em uma experiência de RPG com progressão, classes, loot, equipamentos, dungeons, guildas e história.

## Informações planejadas

- Namespace: `riftborn`
- Behavior Pack: `Riftborn BP`
- Resource Pack: `Riftborn RP`

## Estrutura do projeto

- `docs/`: documentação de design, módulos, guia técnico e prompts do projeto.
- `packs/behavior_pack/`: espaço reservado para o behavior pack.
- `packs/resource_pack/`: espaço reservado para o resource pack.
- `assets/`: materiais de apoio, conceitos e referências específicas do projeto.
- `dist/`: arquivos exportados futuramente para testes e distribuição.

Nenhum arquivo funcional de addon foi criado nesta etapa.
