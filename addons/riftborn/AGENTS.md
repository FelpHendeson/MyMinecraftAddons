# Regras para Agents do Riftborn

Este arquivo orienta alterações dentro do projeto `addons/riftborn/`.

## Escopo do projeto

- Nome: Riftborn: Crônicas do Novo Mundo.
- Tipo: Addon para Minecraft Bedrock Edition.
- Tema: RPG, isekai, fantasia medieval e dark fantasy leve.
- Namespace planejado: `riftborn`.

## Regras de trabalho

- Trabalhar apenas dentro de `addons/riftborn/` quando a tarefa for sobre Riftborn.
- Consultar o contexto do projeto em `PROJECT_CONTEXT.md` antes de criar funcionalidades.
- Seguir o escopo definido em `DEVELOPMENT_RULES.md`.
- Registrar mudanças relevantes em `CHANGELOG.md` quando houver implementação.
- Não criar funcionalidades fora do pedido.
- Não misturar ideias ou arquivos de outros addons neste projeto.
- Manter documentação em português.
- Não criar manifests, itens, receitas, texturas, scripts ou mobs sem solicitação explícita.

## Direção de desenvolvimento

O Riftborn deve evoluir de forma incremental. A base técnica vem antes de sistemas complexos, e cada versão deve ter um objetivo pequeno, testável e documentado.
